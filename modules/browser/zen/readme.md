# browser / zen

zen browser (twilight build) with custom policies and chrome.

**Preference key:** `preferences.modules.browser.zen`
**Default:** `true`

**Files:**
- `home.nix` — zen-browser with privacy policies, transparent browser support, custom user profile; desktop entries for youtube-music, zen, youtube, telegram, google-tasks
- `assets/icons/` — custom svg icons for desktop entries
- `assets/zen-chrome/` — custom browser chrome css

**Dependencies:** `zen-browser` flake input
