# widgets / sherlock

sherlock desktop application launcher.

**Preference key:** `preferences.modules.widgets.sherlock`
**Default:** `false`

**Files:**
- `home.nix` — enables `programs.sherlock` with: app launcher, clipboard execution, calculator, bookmarks, power management, color picker, web search (brave), weather (tbilisi)
- `assets/style.css` — sherlock theme

**Dependencies:** none
