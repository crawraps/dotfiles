# yazi

yazi terminal file manager with catppuccin theme and plugins.

**Preference key:** `preferences.modules.yazi`
**Default:** `true`

**Files:**
- `home.nix` — yazi with zsh integration, smart-enter/yank-file/paste-file/hide-tab-bar plugins, catppuccin mocha-teal flavor, custom keymaps, custom theme
- `assets/init.lua` — yazi init script

**Dependencies:** none
