# kde-connect

kde connect for phone-computer integration.

**Preference key:** `preferences.modules.kdeConnect`
**Default:** `true`

**Files:**
- `system.nix` — opens firewall ports (1714–1732 TCP/UDP) for kde connect
- `home.nix` — enables kdeconnect home-manager service
- `assets/` — kde connect assets

**Dependencies:** none
